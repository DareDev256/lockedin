Embed LockedIn on Your Site

Add a Follow All widget to any website with a single script tag. Your visitors can follow you everywhere, save your contact, and view your full profile.

1 Create your LockedIn profile

Head to tapin-bay.vercel.app and fill in your platforms. Once done, copy the full URL from your browser's address bar.

The URL looks like this:

https://tapin-bay.vercel.app#eyJuIjoiSmFtZ...

The part after the # is your profile hash. That is the value you need for the widget.

2 Add the script tag

Paste the following into your HTML, replacing YOUR_PROFILE_HASH with the value from step 1:

<!-- LockedIn Widget (button mode) -->
<script
  src="https://tapin-bay.vercel.app/widget.js"
  data-profile="YOUR_PROFILE_HASH"
></script>

This renders a floating pill button in the bottom-right corner. Clicking it opens a mini card with your profiles and a Follow All button.

3 Choose your mode

Attribute Values Description
data-profile Base64 string Your encoded profile hash (required)
data-mode "button" | "inline" Floating button (default) or inline card

Button mode (default)

A fixed-position pill button that expands into a card on click. Best for portfolios, blogs, and landing pages.

<script
  src="https://tapin-bay.vercel.app/widget.js"
  data-profile="YOUR_PROFILE_HASH"
></script>

Inline mode

Renders a card directly in the page where the script tag is placed. Best for about pages, contact sections, or sidebars.

<script
  src="https://tapin-bay.vercel.app/widget.js"
  data-profile="YOUR_PROFILE_HASH"
  data-mode="inline"
></script>

? What the widget does

  • Displays your name, handle, bio, and avatar (pulled from GitHub if linked)
  • Follow All opens each of your platform profiles in sequence (600ms stagger to avoid popup blocking)
  • Individual platform icons link directly to your profiles
  • Save Contact generates and downloads a vCard (.vcf) file with all your links
  • Links to your full LockedIn profile page

! Technical details

  • Zero dependencies, vanilla JavaScript (<15KB)
  • Self-contained CSS with .lkin- prefix; will not conflict with your styles
  • Dark glassmorphism design; looks great on any background
  • Responsive, works on mobile
  • No cookies, no tracking, no external requests except Google Fonts and GitHub avatars

Quick test

Create your profile at LockedIn, copy the hash from the URL, and paste this into any HTML file:

<!DOCTYPE html>
<html>
<body>
  <h1>My Site</h1>
  <script
    src="https://tapin-bay.vercel.app/widget.js"
    data-profile="YOUR_PROFILE_HASH"
  ></script>

</html>
